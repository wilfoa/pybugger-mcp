## v0.1.0 (2026-01-14)

### Features

- Initial release of Python Debugger MCP
- MCP server with 24 debugging tools
- Full DAP (Debug Adapter Protocol) support via debugpy
- Session management with concurrent session support
- Breakpoint management with persistence and conditions
- Watch expressions for tracking values across steps
- Program output capture (stdout/stderr)
- Session recovery after server restart
- HTTP REST API for non-MCP clients
- Comprehensive test suite (unit, integration, e2e)
